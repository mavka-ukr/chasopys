# Будуємо Мавку на GNU/Linux

Як збудувати <subject>Мавку</subject> на Linux з джерельних файлів? Читайте далі.

## Встановлення залежностей

Нам будуть потрібні **Ціль**, **Clang** та **Git**.

Встановлення **Clang** та **Git** на **Ubuntu Linux**:

```shell
sudo apt install git clang
```

Встановлення **Clang** та **Git** на **Arch Linux**:

```shell
sudo pacman -S git clang
```

Після встановлення **Clang** та **Git** треба завантажити та встановити **Ціль**.

В [репозиторії <subject>Мавки</subject>](https://github.com/mavka-ukr/mavka) треба дізнатись необхідну версію
<subject>Цілі</subject> та завантажити її з [Архіву Цілі](https://архів.ціль.укр).

Після завантаження, архів необхідно розпакувати та скопіювати звідти виконуваний файл `ціль` в теку `/usr/local/bin`

Далі треба перевірити чи <subject>Ціль</subject> встановлено. Для цього достатньо у вашому терміналі прописати команду
`ціль`: якщо виведе
версію і додаткову інформацію - все добре.

## Будування

Клонуємо репозиторій <subject>Мавки</subject>:

```shell
git clone https://github.com/mavka-ukr/mavka --depth=1
```

Відкриваємо клоновану теку:

```shell
cd mavka
```

Компілюємо файли <subject>Цілі</subject>:

```shell
TSIL="ціль" sh external/scripts/prepare.sh linux-x86_64
```

Переходимо в підготовлену теку:

```shell
cd build/linux-x86_64
```

Будуємо виконуваний файл:

```shell
sh build.sh
```

Пробуємо:

```shell
./out/мавка
```

Для зручності, <subject>Мавку</subject> можна встановити глобально:

```shell
sudo cp ./out/мавка /usr/local/bin
```
